# Tic-Tac-Toe Intelligent Agent 

## 1. Description
This project is an implementation of a Tic-Tac-Toe game using an intelligent agent. The agent is designed to observe the board state, analyze possible moves, and select the best action to maximize the chance of winning or ensuring a draw in the worst-case scenario.

## 2. AI Features
* **Agent Type:** Utility-Based Agent.
* **Algorithm:** Minimax Algorithm (to evaluate possible outcomes and choose the move with the highest utility value).
* **Environment Analysis (ODESA):** Fully Observable, Strategic, Sequential, and Static.

## 3. Project Structure
* **main.html:** The main game interface and dashboard.
* **main.css:** The styling, layout, and responsive design (Dark/Light mode).
* **main.js:** The core game logic and the Minimax AI implementation.

## 4. PEAS Analysis
* **Performance Measure:** Winning the game, minimizing moves, and preventing the opponent from winning.
* **Environment:** 3x3 game board grid.
* **Actuators:** Placing X or O marks in empty cells.
* **Sensors:** Observing the board configuration and occupied positions.
